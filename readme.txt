Converts your default Wordpress editor into a markdown editor.

== Description ==
Converts your default Wordpress editor into a markdown editor. This is meant to be a simple solution - no frills. If you need to switch back, your markdown code will be converted back into regular HTML!

== Installation ==
__From your WordPress dashboard__

Visit ‘Plugins > Add New’
Search for ‘barkdown’
Activate barkdown from your Plugins page.

__From zip__

Upload file
Activate plugin

== Frequently Asked Questions ==
__#1 Will this plugin convert my existing content into markdown?__
- No, this will only convert markdown created while the plugin is active back into HTML.
